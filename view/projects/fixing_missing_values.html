Finding & Fixing
Missing Values

Fixing missing values means, first finding "NaN / Na / nan / None" value.
And, replace those missing values with suitable value.

## Several ways to fix missing values while preprocessing the data ðŸš€
* Fill missing values with Median or Mean/Average value
* Can use Forward fill or Backward fill technique
* Can use Interpolate method
* Remove rows, if there is less no. of rows with missing values.
* Remove column, if that column with missing values won't be requied in analysing the data.
* Fill missing values with a constant value

## Some panda's methods to find/view missing values (NaN, None) ðŸš€
* .isna().sum()
* .isnull().sum()
* .unique() => It doesn't work as desired, for the numeric column.
* .value_counts()

## Some panda's methods to fix missing values ðŸš€
* .fillna()
Examples:
.fillna('1', inplace=True) # Fill all Nan values with 1

* .dropna()
Examples:
df.dropna() # Drops rows containing NaN values
df.dropna(axis=1) # Drops columns containing NaN values

* .replace()
Examples:
df.replace(np.nan, 0) # Replace NaN with 0
